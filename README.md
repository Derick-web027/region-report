# 区域分析报告生成器

一个纯静态网页：选择省份与城市，一键下载该地区的「城投区域分析报告」（Word 文档）。

报告涵盖区域经济、财政与政府债务、土地与房地产市场，骨架与样式统一。所有数值、同比增速、累计跌幅、省内排名、二手房冰山指数跌幅均由数据库自动计算；「支柱产业 / 龙头企业 / 战略定位」等定性内容以【待补充】占位。

## 在线使用

发布到 GitHub Pages 后，访问 `https://<用户名>.github.io/<仓库名>/` 即可。

## 目录结构

- `index.html` — 页面（选区域 → 下载）
- `manifest.js` — 省份/城市索引与文件映射
- `reports/` — 预生成的各城市报告（rNNNN.docx）
- `.nojekyll` — 关闭 GitHub Pages 的 Jekyll 处理

数据来源：企业预警通、中指数据库、Wind 等，仅供研究参考。
